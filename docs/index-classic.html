Transparent by design

LUFSBar is a small app that does one thing. You can verify everything it does.

設計から透明

LUFSBarは1つのことだけをする小さなアプリです。その挙動はすべて検証できます。

No network, no telemetry通信なし・テレメトリなし

Audio is analyzed in memory and never recorded to disk. No accounts, no tracking, no requests.音声はメモリ上で解析されるだけで、録音・保存は一切しません。アカウント不要・追跡なし。

Standards-compliant metering標準規格準拠の計測

ITU-R BS.1770-4 / EBU R128 via libebur128: K-weighting, gating, 4× oversampled True Peak.libebur128によるITU-R BS.1770-4 / EBU R128準拠。K特性・ゲーティング・4倍オーバーサンプリングTrue Peak。

macOS will ask once for System Audio Recording permission — that's the only access LUFSBar needs. It also registers as a login item on first launch so your meter is always there; turn it off anytime and it will never re-enable itself. 初回に一度だけ「システムオーディオ録音」の許可を求めます。LUFSBarが必要とするのはそれだけです。また初回起動時にログイン項目へ登録されます（メーターを常に使えるように）。いつでもオフにでき、勝手にオンへ戻すことはありません。
