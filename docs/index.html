Transparent by design.

LUFSBar is a small app that does one thing. You can verify everything it does.

設計から、透明。

LUFSBarは1つのことだけをする小さなアプリです。その挙動はすべて検証できます。

No network. No telemetry.通信なし。テレメトリなし。

Audio is analyzed in memory and never recorded to disk. No accounts, no tracking.音声はメモリ上で解析されるだけで、録音・保存は一切しません。

One permission only求める許可は1つだけ

macOS asks once for System Audio Recording. That's all LUFSBar needs.初回に一度だけ「システムオーディオ録音」の許可を求めます。必要なのはそれだけ。

100% open source完全オープンソース

MIT licensed. Read the source, build it yourself, or file an issue.MITライセンス。ソースを読むのも、自分でビルドするのも自由。

LUFSBar registers as a login item on first launch so your meter is always there. Turn it off anytime — it will never re-enable itself. 初回起動時にログイン項目へ登録されます（メーターを常に使えるように）。いつでもオフにでき、勝手にオンへ戻すことはありません。
